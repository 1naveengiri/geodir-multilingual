== Description ==

Allows running your directory fully multilingual with GeoDirectory and WPML.

= Key Features =

* Translates all GeoDirectory listings
* Translates GeoDirectory listing taxonomies(categories & tags)
* Translates listing custom fields
* Translates GeoDirectory options

== Installation ==

1. Upload 'geodir-multilingual' directory to the '/wp-content/plugins/' directory
2. Activate the plugin "GeoDirectory Multilingual" through the 'Plugins' menu in WordPress

= Minimum Requirements =

* WordPress 4.9 or later
* GeoDirectory 2.0.0 or later
* WPML Multilingual CMS 3.4 or later
* WPML String Translation 2.0 or later
* WPML Translation Management 2.2 or later
* WPML Media 2.1 or later
